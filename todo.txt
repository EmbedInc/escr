Function RUNEX - Runs program, returns exit status code.

Function RUNTF - Runs program, TRUE if exit status code 0.

Function RUNSO - Runs program, returns first standard output line.

Command APPEND - Appends to end of string.

LOOP DIR - Loops over directory entries.

Function VNL - Expression value ignoring local symbols of this block.

Command FUNCTION - Define a new function.

Command COMMAND - Define a new command.
